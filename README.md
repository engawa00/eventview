# Windows Wake Event Viewer

Windowsのイベントログから、PCがスリープから復帰した履歴とその「復帰理由」を簡単に確認できるツールです。
Tkinterベースの直感的なGUIと、自動化やスクリプト連携に便利なCLIの両方に対応しています。

## 特徴
- **スリープ復帰履歴の表示**: イベントビューアーを開かなくても、簡単に復帰日時と理由を一覧化できます。
- **ハイブリッドなUI**: 引数なしで実行するとGUIが立ち上がり、引数を渡すとCLIツールとして動作します。
- **カレンダー検索**: GUIでは使いやすいカレンダーUIから期間を指定してログを検索できます。
- **外部ライブラリ不要**: Pythonの標準ライブラリ（`subprocess`, `xml.etree.ElementTree`, `tkinter` など）のみで動作し、追加モジュールのインストールは不要です。

## 動作要件
- OS: Windows 11
- Python: 3.6以降 (Tkinterが有効になっていること)

## 使い方

### GUIモードで起動
コマンドプロンプトまたはPowerShellから引数なしで実行すると、GUIウィンドウが立ち上がります。
```powershell
python event_viewer.py
```
画面上部の「📅」ボタンからカレンダーポップアップを開き期間を指定して「検索」ボタンを押すことで、リストに復帰ログが表示されます。行をクリックすると下段のテキストボックスに復帰理由の全文が表示されます。

### CLIモードで起動
期間を引数で指定することで、指定期間内のログをコンソール出力します。
```powershell
# 全期間を取得 (指定なし)
python event_viewer.py

# 特定の期間を指定して取得
python event_viewer.py --start 2026-01-01 --end 2026-03-31
```
